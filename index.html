Para que serve o HTML?

Voltar

O HTML, ou HyperText Markup Language — Linguagem de Marcação de Hipertexto não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira. As tags anexas podem transformar uma palavra ou imagem num hiperlink, pode colocar palavras em itálico, pode aumentar ou diminuir a fonte e assim por diante. Veja mais sobre o assunto clicando aqui.

O que são TAGS?

Voltar

No HTML a tag é usada para criar um elemento. O nome de um elemento HTML é o nome usado entre colchetes angulares como "< i >" para deixar o texto em itálico. Note que ao fechar uma tag ela tem uma barra antes de seu nome, "< / i >". Em algumas tags, o fechamento delas não é necessário, fazendo parte da sintaxe delas mesmas.

Listas ordenadas e não ordenadas

Voltar

As listas ordenadas são as listas que contém uma sequência numérica dos itens listados, onde você define a sequência utilizada. Como por exemplo:

Top 3 comidas brabas:

  1. Arroz
  2. Feijão
  3. Carne

Já as listas não ordenadas são as listas que citam alguns itens, mas não necessáriamente precisam de uma sequência. Como por exemplo:

Lista de compras:

  • Arroz
  • Feijão
  • Carne
